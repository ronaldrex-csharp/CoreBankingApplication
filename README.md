# CoreBankingApplication

CoreBankingApplication is a secure banking system built with ASP.NET Core, Entity Framework Core, SQL Server, and ASP.NET Core Identity. The application allows users to manage accounts, perform banking transactions, and securely authenticate through a modern web interface.
